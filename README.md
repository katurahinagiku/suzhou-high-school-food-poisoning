**号外！号外！苏州中学脸都不要了。是这样的，本日苏州中学于上午大课间举行校领导脸皮大甩卖，往日卖给家长，说这儿教育资源好，人才辈出，老师优秀负责，可被某些似人非人的领导以为是在反讽自己，变成前些日子的卖给学生的模式，说这儿学生自由度高，作业少，中午吃饭能去食堂、能点外卖（2025年12月29日前）、能出门吃（疫情前），运动也不差，各个身强体壮，最关键是成绩没落下！可以看出这儿简直是学生的天堂，而我要说这座天堂其实是座城堡，城外的人想冲进去，城里的人想冲出来，这又怎么回事呢？还是似人非人的领导认为自己太不像人，于是乎对标其他学校学苏高般开始学起别的学校，诸如提升一下清北人数、985/211、竞赛成绩之类，但有的事没办法，傻子不说拜师，学正常人都只能是邯郸学步，不能说它们学不会，得说它们学了个四不像，就像让它们学分辨食物，别人说像木头这么硬的东西怎么能吃，它们便知道了：哦，软的东西才适合吃，于是找来一盘苏膳集团的米饭与一滩携有诺如病毒的粪便，还以为这两样入口即化、味道细腻之物能吃，估计是吃完后变成了“化物”，改号中年油腻大妈/大叔了，装作品尝到了人间至臻，实则自己也吃不出好坏，只知道“硬的不能吃”。到此为止停手就算了，化物们还以为其他人也能吃苏膳膳食与诺如排遗，它们喂给同学们吃，不吃，便苦口婆心道“你不尊重我，我也不尊重你”，称“天将降大任于斯人也，必先‘喂其粪便’”，那样还不如“饿其体肤”呢！领导们看来，我们是一群孙猴子，缺乏它们这样贤明的唐僧的管教；而我们看来，苏高中被一群双商皆无的人鬼混血种支配了，其实他们与人相关的也只有外貌上的相似了，太像人类了，导致我们看着它们还得起到恐怖谷效应。**

值得注意的是，去年春天，学校东校区（正谊校区）食堂也曾出现食品安全问题。（盖饭中烟也）

## Overview

### 3月12日，江苏省苏州中学早上有大量学生出现肚子痛、腹泻、呕吐等症状而请假送医，共计有上百人。

### 大量学生的医院化验单显示为细菌感染，而学校将此次事件定性为诺如病毒传播，当天中午提前放学，称“为了切断病毒传播”，当天下午及第二天全天线上上课，并再三提示家长是病毒感染，且不要传播相关信息。

### 事后学校食堂被中国卫生取样检查，至今未公布结果，大量学生回家后也产生了前文细菌感染的症状。

### 有学生和家长对通报结果有异议或传播相关信息，均被单独联系删除相关投稿。

### 在当天下午的线上教学中，该校某老师微信聊天甚至显示出当地公安副局长单独发送家长个人信息给老师，“对通报结果有异议”“家校协同高效配合”。

### 学校食堂从去年8月1日开始由江苏苏膳膳食集团有限公司运营。学校方面目前对此事无回应、无处理。
